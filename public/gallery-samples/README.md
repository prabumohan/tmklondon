# Gallery sample images (for Admin upload)

These JPEGs match the site’s demo gallery sources. They are generated by:

```bash
npm run download:gallery-samples
```

**Admin → Gallery:** choose the category tab (Teachers, School, Events, Community), then use **Choose files** or drag-and-drop the `.jpg` files from the matching subfolder here:

| Folder | Upload when category is |
|--------|-------------------------|
| `teachers/` | எங்கள் ஆசிரியர்கள் |
| `school/` | பள்ளி நடவடிக்கைகள் |
| `events/` | நிகழ்வுகள் |
| `community/` | சமூக கூட்டங்கள் |

You can select all three images in a folder at once for that category.

*Optional:* This folder is under `public/` so you can also open files from disk at  
`public/gallery-samples/...` in your project.
