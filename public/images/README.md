# Images Directory

## Carousel Images

Place your hero carousel images in `public/images/carousel/`:

- `tamil-heritage-1.jpg` - First carousel image
- `tamil-heritage-2.jpg` - Second carousel image  
- `tamil-heritage-3.jpg` - Third carousel image

**Recommended specifications:**
- Format: JPG or PNG
- Dimensions: 1920x1080px or larger (16:9 aspect ratio)
- File size: Optimized for web (< 500KB each)
- Content: Tamil heritage, cultural events, school activities, community gatherings

## Logo

Place your logo at:
- `public/logo.png`

**Recommended specifications:**
- Format: PNG with transparent background
- Dimensions: 200x200px or larger (square)
- File size: < 100KB

## Gallery Images

For the gallery page, you can organize images in:
- `public/images/gallery/`

## Other Images

- Event photos: `public/images/events/`
- School photos: `public/images/school/`
- Publications covers: `public/images/publications/`
